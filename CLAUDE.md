# Book Reader Repository

This repository is designed to help you learn from PDF technical books using Claude Code.

## Current Content

**"The Ultimate Guide to React Native Optimization (2026)"** by Callstack - A comprehensive resource covering performance optimization across JavaScript, native, and bundling layers in React Native applications.

## Structure

- `*.pdf` - Book files in the repository
- `.claude/skills/` - Specialized skills for working with specific books

## Getting Started

Ask Claude to help you learn from the book. Claude will automatically invoke the appropriate skill to assist you with:
- Reading and extracting content from the PDF
- Explaining concepts and chapters
- Quizzing you on the material
- Summarizing sections
- Connecting ideas across chapters
